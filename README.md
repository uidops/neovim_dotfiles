## requirements
``` text
neovim   - https://github.com/neovim/neovim
vim-plug - https://github.com/junegunn/vim-plug
```


## install

```bash
$ git clone https://github.com/siruidops/neovim_dotfiles ~/.config/nvim/
$ nvim +PlugInstall
```
